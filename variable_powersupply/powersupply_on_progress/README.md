# Portable Bench Power Supply V2 (In Progress)


* Upgraded current measurement system with a larger shunt to support the converter's full 20A output capability.
* USB-C PD/QC3 input support for operation from compatible chargers and power banks (up to 20V input).
* DC barrel jack input for batteries and external DC power supplies.
* Dedicated USB-C output capable of delivering up to 65W.
* Improved ventilation and thermal management for sustained high-power operation.
* Portable enclosure designed for both bench-top and field use.
